# Aide-mémoire Conformité

## Les 6 règles essentielles

| # | Règle | Limite | Type |
|---|-------|--------|------|
| 1 | Repos quotidien | ≥ 11h entre interventions | ❌ Bloquant |
| 2 | Repos hebdomadaire | ≥ 35h consécutives/semaine | ❌ Bloquant |
| 3 | Durée quotidienne | ≤ 10h/jour | ❌ Bloquant |
| 4 | Durée hebdomadaire | ≤ 48h/semaine | ❌ Bloquant |
| 5 | Pause obligatoire | ≥ 20 min si > 6h | ⚠️ Avertissement |
| 6 | Chevauchement | Interdit | ❌ Bloquant |

## Majorations salaire

| Condition | Majoration |
|-----------|------------|
| Dimanche | +30% |
| Jour férié habituel | +60% |
| Jour férié exceptionnel | +100% |
| Heures de nuit (21h-6h) | +20% |
| Heures sup (1-8h) | +25% |
| Heures sup (> 8h) | +50% |

## Pauses recommandées

| Durée | Pause |
|-------|-------|
| < 4h | 0 min |
| 4-6h | 15 min |
| 6-8h | 20 min |
| 8-10h | 30 min |
| > 10h | 45 min |

## Calcul rapide

```
Salaire = (Heures - Pause) × Taux × (1 + Majorations)
```

## Erreurs fréquentes

1. **Enchaîner sans repos** → Respecter 11h minimum
2. **Semaine trop chargée** → Max 48h lundi-dimanche
3. **Journée trop longue** → Max 10h effectives
4. **Oublier les pauses** → Obligatoire dès 6h
